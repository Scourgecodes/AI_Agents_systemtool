# systemtool

An interactive command-line Python agent designed to retrieve local system information and display directory file details safely.

## Features

* **OS Info:** Displays the operating system name, release version, and computer architecture.
* **Python Version:** Checks and prints the exact version of Python currently running.
* **Current Path:** Displays the exact folder path where the script is located on your computer.
* **List Files:** Scans and lists all files inside the active folder directory.

## Technology Stack
* **Language:** Python 3
* **Core Built-in Modules:** `os`, `sys`, `platform`
